Конструктор UtilityClass():
1) aload_0 загружает ссылку this на текущий объект в стек.
2) invokespecial #1 вызывает конструктор суперкласса (java/lang/Object."<init>":()V), чтобы инициализировать объект.
3) return завершает выполнение метода.

Метод getStringLength(java.lang.String):
1) aload_1 загружает первый параметр метода (входную строку) в стек.
2) ifnull 9 проверяет, равна ли загруженная ссылка null. Если да, переходит к инструкции под номером 9.
3) invokevirtual #7 вызывает метод length() для объекта String, чтобы получить его длину.
4) ireturn возвращает целочисленное значение из стека (длину строки).
5) iconst_0 помещает в стек число 0.
6) Если строка была null, исполнение переходит к iconst_0, и затем ireturn возвращает 0.

Метод callMethodOnObject(java.lang.Object, java.lang.String):
1) Получает класс объекта и ищет метод по имени и параметрам.
2) Вызывает этот метод и возвращает результат, приведенный к типу String.
3) В случае исключения выводит стек вызовов и возвращает null.

Метод changeObjectField(java.lang.Object, java.lang.String, java.lang.String):
1) Получает класс объекта, находит указанное поле по имени, делает его доступным и устанавливает новое значение.
2) Обрабатывает исключения, выводя стек вызовов.
